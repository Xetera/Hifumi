## Setting up Hifumi

> Possibly outdated

1. run: `git clone https://github.com/moedevs/hifumi.git`
2. copy `.env.example` to `.env` and add your bot token
3. install docker-compose
4. run: `cd hifumi`
5. run: `docker-compose -d up`
6. run: `npm install`
7. run: `npm run codegen`
8. run: `npm run dev`
